Shortcuts

  • Enter/Ctrl + Enter to show next/previous suggested sample to annotate. Works if a model that supports suggestions is active.
  • Left/Right to override sample class when focused on a data sample.
  • Keys 0-9 are decoded into one of the alphabetically ordered classes and assigned to an active data sample.

Tools and notes

  • Use Set cursor size to change nearest neighbors sampling. If > 1 turns into KNN
  • You have an option to show your model's predictions if it is active. Toggle Show predictions.
  • Use Filter tool to apply class/prediction/etc filter. Also, select binary operators to compare class and prediction. Use Reset button to show everything (default).
  • For Clustering options please refer to the docs in core.clustering.py
