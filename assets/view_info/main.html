Shortcuts

  • Ctrl + left/right to show next/previous image.
  • Esc to focus on tag search bar.
  • While focused on search bar, press Up/Down to select a tag from the list below.
  • Press Return while in search bar or structure bar to add the tag to the pool.
  • Press Ctrl + Return to add the tag to the pool and assign to the image.
  • To select tag from the pool, double-click on it or focus and press Return.

Tools and notes

  • The working directory is selected through Select Folder button.
  • Images in the directories are then retrieved via rglob, filtered by suffix, and sorted by absolute path.
  • Toggle Edit tags to turn tags into buttons. Press on them to remove the tag from the list of labels.
  • Toggle Preview folder to show thumbnails of images in the same subfolder as the active image. Careful when there are a lot of images.
  • Specify the path in the structure to add tags to the pool in folders.
  • Focus on a tag from the pool, update the structure value and press Move button above to move between folders.
  • Select a recommendation method from the drop-down list on the right below. When models are selected, they are going to be used as feature extractors for image and tags. The ui may freeze while loading.
